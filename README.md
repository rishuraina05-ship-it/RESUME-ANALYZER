# RESUME-ANALYZER
AI-based resume analyzer that extracts text from PDF resumes and provides automated feedback, scoring, and improvement suggestions using NLP techniques.

# AI Resume Analyzer 📄

## Overview
This project analyzes resumes and provides automated feedback based on skills, structure, and content quality.

## Features
- Upload PDF or TXT resume
- Extracts text using PyPDF2
- Scores resume based on key skills
- Provides improvement suggestions
- Clean UI using Streamlit

## Tech Stack
- Python
- Streamlit
- PyPDF2
- Basic NLP logic

## How to Run
1. Install dependencies:
   pip install streamlit PyPDF2

2. Run:
   python -m streamlit run app.py

## Use Case
Helps job seekers improve resumes using automated AI-based analysis.
